Hey Bartender.

Hey Bartender is an elegant web application designed to help you find the perfect cocktail for any moment. By taking in key information about the user; Hey Bartender generates a list of relevant cocktails for that particular instance. Each cocktail entry is garnished with a high quality image, an ingredients list, and professional instructions on how to prepare the beverage yourself.

The application was built for my Senior Capstone project and has won the most popular project within the Computer Science and Information Science department that year. It uses Boostrap, PHP, AJAX, and MySql. Bootstrap controls the UI layer of the application, AJAX communicates between the UI and application layer and PHP is used for querying the database and dynamically creating the HTML code for the personalized response.
